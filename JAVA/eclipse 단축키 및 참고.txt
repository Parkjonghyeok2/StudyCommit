Alt + 방향키(위/아래) - 현재 커서가 있는 줄의 소스를 한칸 위/아래로 이동시킵니다.
Ctrl + Alt + 방향키(위/아래) - 이 단축키는 현재 커서가 있는 한줄을 위/아래로 복사하는 단축키입니다.
Ctrl + D - 커서에 있는 한줄을 모두 지우는(날려버리는) 명령어입니다.
Ctrl + Shift + F - 문단을 선택후 단축키를 실행하면 이클립스에서 자동으로 정렬이 됩니다.
Ctrl + I - 이 단축키 또한 자동으로 정렬해주는 단축키입니다.
Ctrl + 마우스 좌클릭 -  이 단축키는 클래스, 메소드 등에대해 확인하고 싶을때, 사용하는 단축키입니다.
Ctrl + Shift + C - 커서가 선택된 라인 혹은 지정한 영역 전부를 주석 처리 합니다.
Ctrl + Sfhit + W - 열려있는 모든 창을 닫는 단축키입니다.(마우스 우클릭 - Close All 과 동일합니다.)
Ctrl + W - 현재 열려있는 창 하나를 닫습니다.
Ctrl + Shift + O - 이것도 굉장히 자주 사용하는 단축키 중 하나입니다. 자동으로 import를 해주는 단축키입니다.
Ctrl + Shift + L - 이 단축키 또한 굉장히 자주 사용하는 단축키입니다. 이클립스의 모든 프로젝트에서, 찾고자하는 코드가 있을경우 정말정말정말 유용합니다.
